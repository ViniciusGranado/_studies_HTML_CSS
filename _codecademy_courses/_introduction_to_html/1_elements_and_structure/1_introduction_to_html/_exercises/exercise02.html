To-do list:

Things to do before guests arrive.

  • Change the toilet paper.
  • Check if food is placed properly.
  • Check if the audio system is ready. If not, follow the instructions below this list.
  • Check if coffee is still warm.

Audio system instructions:

If the audio system is not turned on, follow the instructions below:

  1. Check if the power cord is correctly plugged.
  2. Press the big red button on the left side.
  3. Wait for the three green lights to turn on.
  4. Press the play button. Music should start automatically.
